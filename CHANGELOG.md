# Changelog

All notable changes to Speedboat will be documented here.

---

## v1.1.0 — 2026-05-05

Documentation and model clarification update.

- Reframed the operating model as three work lanes (Preview, Build, Run) plus the Land outcome layer
- Updated the README, full model, one-page summary, leadership brief, and lane guide to clarify Landings vs work lanes
- Simplified Jira setup around a labels-only Kanban board with standard columns and optional swimlanes
- Added guidance for team size and boat boundaries across the docs
- Expanded the FAQ with guidance for larger teams and piloting without org redesign
- Updated onboarding and survey wording to match the three-lanes-plus-outcome model
- Tightened adoption guidance with a minimum viable rollout path and clearer tooling caveats
- Restored the trial evaluation template so the end-of-trial path is complete
- Corrected the LICENSE file name

## v1.0.0 — 2026-05-01

Initial public release.

- Operating model: three work lanes (Preview, Build, Run) plus the Land outcome layer, weekly steering rhythm, fortnightly Route Planning
- Guides: getting started checklist, ceremony guides (Set Course, Course Check, The Landing, Route Planning), facilitation rotation, Which Lane decision tree
- Templates: Landing Log, Weekly Snapshot, Preview Decision Record, Trial Evaluation, Team Survey
- Documentation: full model, one-page summary, leadership brief, FAQ and troubleshooting
